# Documentación por Módulos

Este directorio agrupa especificaciones y decisiones de arquitectura para cada módulo del monolito Angelow 2.

- `catalog.md`
- `sales.md`
- `customers.md`
- `backoffice.md`
- `notifications.md`
- `content.md`
- `support.md`

Cada archivo debe incluir:
1. Objetivo del módulo y stakeholders.
2. Límites del dominio y entidades clave.
3. Tablas / migraciones asociadas.
4. Integraciones externas.
5. Lista de endpoints (HTTP/API) y eventos.
6. Estrategia de pruebas.
